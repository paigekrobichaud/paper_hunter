# Paper Hunter

Research paper semantic search engine.

## Description

Given a search query, Paper Hunter finds and returns the most relevant sections of research papers, along with the paper title and a semantic similarity score.

Current features:
- PDF ingestion from the `papers/` directory
- Text extraction
- Document and Chunk abstractions
- Sentence-transformer embeddings
- Semantic search using cosine similarity
